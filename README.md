# Alan Hu - Personal Website

A modern, creative personal website showcasing my journey, experience, and projects.

## Features

- **Hero Section**: Animated introduction with gradient backgrounds
- **Story Section**: Personal narrative about my upbringing and transformation
- **About Section**: Interests and passions in AI/ML, software engineering, and low latency systems
- **Experience Timeline**: Professional experience with visual timeline
- **Projects**: Featured projects including Locus VR and GPT4SME
- **Skills & Education**: Technical skills, education, certifications, and awards
- **Contact**: Easy ways to get in touch

## Tech Stack

- Next.js 12
- React 18
- Chakra UI
- Framer Motion
- React Icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
npm start
```
